# Manual Test Scenarios

## Purpose

This folder contains manual test scenarios used to validate milestone behavior. Each milestone folder provides:
- a README that explains how to run the scenarios
- a scenario index
- per-scenario steps and expected outcomes

## How to Use

1. Choose the milestone that matches the behavior you need to validate.
2. Read that milestone's README for setup, scope, and validation checklist.
3. Run each scenario in order and record results in the milestone's "Test Run Log."

## Which Milestone Is Current

If multiple milestone sets exist, prefer the newest milestone folder (highest number/letter) unless a specific test plan says otherwise.
