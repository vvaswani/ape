# A.P.E aka Automated Portfolio Evaluator!

Meet APE — your AI-native portfolio advisor!

## Start

* `export GOOGLE_GENERATIVE_AI_API_KEY=...` # or `OPENAI_API_KEY`, `ANTHROPIC_API_KEY`
* `docker compose up`
* Browse to `http://localhost:3000`
